> 항상 로드하는 도메인 핵심 규칙. 모든 포스트는 이 문서를 준수해야 합니다.

## 1. YMYL 등급 — Money 최고

보험은 Google YMYL "Your Money" 최고 등급. 잘못된 정보는 금전적 손해 + 보장 공백 + 법적 리스크로 직결.

## 2. 보험업법 / 광고 규제 (한국)

이 사이트는 **"보험대리점/설계사"가 아닌 "보험 정보 큐레이터"** 로 명확히 포지셔닝.

### 절대 금지 표현
- "원금보장" (연금/저축보험도 금지 — 공시이율 변동성)
- "손해 없음", "확정수익률" (변액보험 확정 표현 금지)
- "반드시 가입해야 한다", "꼭 필요하다" (권유 단정)
- "최고의 보험", "무조건 1등" (비교광고 금지 준용)
- 특정 설계사·대리점 추천 서술
- "환급률 100% 이상" (해지환급금 기준 명시 없이)
- 보험계약 체결·모집 유도 표현

### 권장 표현 전환
- "가입 전 반드시 확인할 것" → "검토 시 고려할 사항"
- "보장 최고" → "보장 범위가 넓은 편"
- "저렴한 보험료" → "보험료 수준이 낮은 편" / "업계 평균 대비"
- "추천" → "비교 시 고려 가능"

### 필수 고지
- 모든 페이지 푸터(자동): 본 사이트는 보험 상품을 판매·모집하지 않으며, 게재 정보는 일반적 설명입니다.
- 보험료 예시 옆: "공시 기준일 / 가입 조건(성별·연령·보장기간)" 병기 필수
- `/about/` `/editorial-policy/` 에 광고 수익 운영 명시

## 3. 정확성 등급 (Level A/B/C)

**Level A — 개별 보험상품 실제 조건 (가장 엄격)**
- 특정 상품 보험료·보장한도·면책·갱신
- 약관 해석 / 청구 절차 / 분쟁 사례
- → **공시실 출처 필수**, "최종 가입 전 약관·상품설명서 확인" 면책 고정 노출

**Level B — 보험 종류·제도 (중간)**
- 실손이란 / 자동차보험 대인배상 이란
- 청구 일반 절차 / 세액공제
- → 금감원·협회 공식 자료 우선, 일반적 설명으로 한정

**Level C — 보조 생활 정보 (완화)**
- 운전자보험 검토자가 참고할 블랙박스 정보 등
- → 커머스 연계 가능 (쿠팡 파트너스 링크 전용 영역)

## 4. 보험 표기 준수

- 보험료는 반드시 "{성별} {연령} 기준, {공시일}" 함께 표기
- 상품명은 정식 상품명 그대로 (별칭 시 정식명 병기)
- 갱신형 / 비갱신형 / 만기환급 / 소멸성 구분
- 순보험료 / 영업보험료 혼동 방지
- "월 만원대부터" (×) → "최저 월 {금액}원, {조건 기준}" (○)

## 5. E-E-A-T

### Experience
- 청구 사례는 "실제 보험소비자연맹/분쟁조정 공개 사례"만 인용
- 개인 경험은 최소화

### Expertise
- 사이트 운영자는 "보험설계사 자격 없음 / 보험 정보 큐레이터"로 명확히 포지셔닝
- **"보험전문가 감수" 허위 표기 절대 금지**
- 공식 자료 출처 2개 이상 필수:
  - 금감원 금융상품통합비교공시 (finlife.fss.or.kr)
  - 생명보험협회 공시실 (klia.or.kr)
  - 손해보험협회 공시실 (knia.or.kr)
  - 각 보험사 공시실

### Authoritativeness
- 공식 기관 아웃바운드 링크 필수
- 출처 섹션에 URL + 기관명 + 참조일자
- 특정 블로그 / 카페 / 유튜브 인용 금지

### Trustworthiness
- 각 페이지 업데이트 일자
- `/disclaimer/` 보험 특화 별도 작성
- 광고 수익 운영 고지
- 연락처 실존(오류 신고 창구)
